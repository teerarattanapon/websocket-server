# websocket-server
node.js websocket for testing simulate data to plot real-time graph.

## **Requirement**
ws:Node.js WebSocket library ([Link](https://github.com/websockets/ws#simple-server))

```
npm install --save ws
```
## **Usage**
1. Start websocker server

```
node ws-server.js
```
2. Run ws-client.html
